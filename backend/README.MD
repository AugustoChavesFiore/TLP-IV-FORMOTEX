

# Backend FORMOTEX

## Servidor construido con una base de datos MongoDB 

1. Instalar dependencias
```Bash
$ npm intall
```
2. Renombrar el archivo `.env.template` a `.env `y completar variables de entorno

3. Ejecutar Proyecto en modo desarrollo
```Bash
$ npm run dev
```
